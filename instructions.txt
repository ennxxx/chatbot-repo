To start the program, run main.

For input with more than one word, please insert a _.

Ex.
What is your main concern? |: muscle_aches.

In inputting the chief complaint, the ChatBot is limited to
take take input defined in its database. 

Here are some examples to start with:
fever
chills
headache
cough
vomiting
muscle_aches
sore_throat
watery_eyes
fatigue

Note: The program uses dynamic predicates that is retracted at 
the beginning and the end of the program to ensure 
proper functionality. Issues might occur when terminating/closing
the program before it has finished running. To prevent this, it is 
recommended to allow the program to continue running until it has 
finished its tasks.
